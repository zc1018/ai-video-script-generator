# Prompt Patterns and Techniques

## Character Description Patterns

### Complete Character Specification Template

```
**角色定义：**
- [角色名称]：[身份]，[主色调][服装类型]，[衬衫颜色和类型]，[配饰1]，[配饰2]，[裤子/裙子描述]，[鞋子描述]，[发型描述]
```

**Example:**
```
**角色定义：**
- 男主角：飞行员父亲，深蓝色飞行员制服外套，白色长袖衬衫，深蓝色领带，金色肩章（四道杠），深蓝色制服裤，黑色皮鞋，短发
```

### Age-Specific Descriptions

**Children (3-10 years):**
- Emphasize size contrast: "迷你版", "明显过大", "袖子太长"
- Include cute details: "歪歪扭扭", "不标准", "奶声奶气"
- Hairstyles: "双马尾", "丸子头", "短发"

**Teenagers (11-18 years):**
- Modern fashion elements
- Casual but styled
- Age-appropriate accessories

**Adults (19-60 years):**
- Professional or casual based on role
- Detailed accessories
- Grooming details (hairstyle, facial hair)

**Elderly (60+ years):**
- Comfortable clothing
- Traditional elements if appropriate
- Dignified appearance

## Scene Description Patterns

### Location + Atmosphere Template

```
场景：[具体地点]，[时间/光线]，[氛围描述]
```

**Examples:**

**Indoor:**
```
场景：明亮的现代机场航站楼大厅，自然光透过大玻璃窗洒进来，繁忙的人流背景
场景：温馨的家庭客厅，柔和的灯光，舒适的沙发和装饰
场景：专业的办公室，明亮的日光灯，整洁的办公桌
```

**Outdoor:**
```
场景：机场停机坪，黄昏时分，金色阳光洒在地面上
场景：城市公园，春日午后，阳光明媚，绿树成荫
场景：海滩，日落时分，橙红色的天空，海浪轻拍沙滩
```

**Vehicle/Transportation:**
```
场景：飞机客舱内，明亮的顶灯，窗外是快速移动的跑道
场景：汽车内部，傍晚，街灯从窗外掠过
场景：火车车厢，白天，窗外风景快速移动
```

## Action Description Patterns

### Basic Action Template

```
动作：[主语] + [动作动词] + [对象/方向] + [方式/情感]
```

**Examples:**

**Movement:**
```
动作：男主角和女主角并肩走来，拉着行李箱，步伐稳健
动作：女儿从后面跑过来，拖着迷你行李箱，兴奋地挥手
动作：三人牵手走向夕阳，背影逐渐远去
```

**Interaction:**
```
动作：母亲蹲下来对女儿讲话，女儿认真点头
动作：父亲把女儿抱起来坐在腿上，温柔地微笑
动作：女儿拿起餐盒，踮起脚尖想递给乘客
```

**Emotional Expression:**
```
动作：女儿紧张地闭上眼睛，紧紧抓住母亲的手
动作：女儿突然睁开眼睛，表情从紧张变成惊喜
动作：父亲和母亲相视而笑，眼神充满温柔
```

## Camera Angle Patterns

### Angle Progression Template

```
镜头：[角度] + [运动（可选）] + [焦点]
```

**Static Angles:**
```
镜头：全景，展现整个场景和所有人物
镜头：中景，聚焦两人对话
镜头：近景，捕捉面部表情
镜头：特写，强调手部动作
镜头：远景剪影，营造氛围感
```

**Dynamic Angles:**
```
镜头：推进，从全景推进到近景，聚焦女儿表情
镜头：拉远，从特写拉远到中景，展现环境
镜头：跟随，跟随女儿跑向父母
镜头：环绕，360度展现三人合影
镜头：俯拍，从上方俯视整个场景
镜头：仰拍，从下方仰视高大的飞机
```

### Shot Sequence Patterns

**Establishing → Detail:**
```
第1段：全景建立场景 → 中景介绍人物 → 近景展现表情
```

**Action → Reaction:**
```
中景：女儿做出动作 → 特写：父母的反应表情
```

**Wide → Tight → Wide:**
```
全景：建立环境 → 特写：关键细节 → 全景：完整画面
```

## Emotional Beat Patterns

### Comedy Patterns

**Setup → Punchline:**
```
Setup (0-6秒): 女儿一本正经地拿起飞行帽
Punchline (6-12秒): 帽子太大，完全遮住眼睛
```

**Contrast:**
```
Serious action + Silly result = Comedy
认真的态度 + 不标准的动作 = 搞笑效果
```

**Repetition with Variation:**
```
Segment 1: 女儿第一次尝试，失败
Segment 2: 女儿第二次尝试，更大的失败
Segment 3: 女儿第三次尝试，意外成功
```

### Drama Patterns

**Tension → Release:**
```
Build (0-6秒): 飞机开始颠簸，女儿紧张
Peak (6-9秒): 颠簸加剧，女儿闭眼
Release (9-12秒): 颠簸停止，女儿松一口气
```

**Anticipation → Surprise:**
```
Anticipation (0-6秒): 女儿期待地看着礼物盒
Surprise (6-12秒): 打开后发现意外的内容
```

### Heartwarming Patterns

**Struggle → Support:**
```
Struggle (0-6秒): 女儿努力够不到
Support (6-12秒): 父母帮助，共同完成
```

**Achievement → Celebration:**
```
Achievement (0-6秒): 完成困难任务
Celebration (6-12秒): 全家庆祝，拥抱
```

## Time-Based Breakdown Patterns

### 12-Second Segment (4 × 3-second beats)

```
**0-3秒：** [建立/介绍]
- 场景：[环境描述]
- 镜头：[通常用全景或中景]
- 动作：[初始状态或进入动作]

**3-6秒：** [发展/转折]
- 镜头：[切换角度或推进]
- 动作：[主要动作开始]

**6-9秒：** [高潮/关键]
- 镜头：[特写或近景捕捉关键时刻]
- 动作：[关键动作或情感爆发]

**9-12秒：** [结束/过渡]
- 镜头：[拉远或切换]
- 动作：[完成动作，为下一段做准备]
```

### 15-Second Segment (5 × 3-second beats)

```
**0-3秒：** [建立]
**3-6秒：** [发展]
**6-9秒：** [转折]
**9-12秒：** [高潮]
**12-15秒：** [结束]
```

### 30-Second Segment (6 × 5-second beats)

```
**0-5秒：** [场景建立]
**5-10秒：** [人物介绍]
**10-15秒：** [冲突/问题]
**15-20秒：** [发展/尝试]
**20-25秒：** [高潮/解决]
**25-30秒：** [结束/过渡]
```

## Dialogue and Sound Patterns

### Dialogue Integration

**Direct Speech:**
```
动作：女儿对着话筒认真说"我们要降落了，请大家坐好"
动作：父亲蹲下来说"好啊，等你长大"
```

**Implied Dialogue:**
```
动作：母亲蹲下来对女儿讲话（暗示对话但不具体说明）
动作：父母交流眼神，会心一笑
```

**Vocal Characteristics:**
```
奶声奶气的发音
认真的语气
兴奋的大喊
温柔的低语
```

### Sound Effects Descriptions

**Ambient Sounds:**
```
背景是繁忙的机场广播声
客舱内轻微的引擎声
海浪拍打沙滩的声音
```

**Action Sounds:**
```
行李箱在地上发出咔咔的声音
飞机引擎轰鸣声
脚步声在走廊回响
```

**Emotional Sounds:**
```
女儿兴奋的笑声
叹气声表达无奈
掌声表达庆祝
```

## Transition Patterns

### Scene-to-Scene Transitions

**Location Change:**
```
Segment N结束：三人走向登机口
Segment N+1开始：飞机客舱内，三人已就座
```

**Time Jump:**
```
Segment N结束：飞机起飞，离开地面
Segment N+1开始：飞行中，已在高空
```

**Perspective Shift:**
```
Segment N结束：驾驶舱，父亲专注驾驶
Segment N+1开始：客舱，母亲和女儿的视角
```

### Emotional Transitions

**Contrast:**
```
Segment N：紧张的起飞
Segment N+1：放松的飞行中
```

**Escalation:**
```
Segment N：小小的搞笑
Segment N+1：更大的搞笑
Segment N+2：最大的搞笑高潮
```

**Resolution:**
```
Segment N：问题/冲突
Segment N+1：尝试解决
Segment N+2：成功解决，庆祝
```

## Style-Specific Patterns

### Cinematic Style

**Characteristics:**
- Longer takes (6-8 seconds per shot)
- Smooth camera movements
- Dramatic lighting
- Atmospheric descriptions

**Pattern:**
```
场景：[详细的环境描述，强调光影]
镜头：[缓慢的运动，如推进或环绕]
动作：[细腻的情感表达，微妙的动作]
氛围：[电影感，戏剧性]
```

### Documentary Style

**Characteristics:**
- Handheld feel
- Natural lighting
- Candid moments
- Realistic actions

**Pattern:**
```
场景：[真实的环境，自然光]
镜头：[手持跟拍，略有晃动]
动作：[自然的互动，非摆拍]
氛围：[纪实感，真实感]
```

### Commercial Style

**Characteristics:**
- High production value
- Perfect lighting
- Polished actions
- Brand-focused

**Pattern:**
```
场景：[完美的环境，专业布光]
镜头：[稳定器运动，专业构图]
动作：[精心设计的动作，展示产品/服务]
氛围：[高端感，专业感]
```

### Social Media Style

**Characteristics:**
- Fast-paced
- Trendy elements
- Relatable moments
- Hook-driven

**Pattern:**
```
0-3秒：[强烈的视觉钩子]
3-9秒：[快速展开，多个快切]
9-12秒：[高潮或笑点]
```

## Color and Costume Patterns

### Color Coordination

**Family Unity:**
```
父亲：深蓝色系（专业）
母亲：酒红色系（优雅）
女儿：酒红色系（与母亲呼应，家庭感）
```

**Contrast for Visibility:**
```
主角：鲜艳颜色（红、蓝、黄）
配角：中性颜色（灰、米、棕）
背景：柔和颜色（不抢主角）
```

**Mood-Based Colors:**
```
温馨场景：暖色调（米色、浅黄、粉色）
专业场景：冷色调（深蓝、灰色、黑色）
欢乐场景：明亮色（亮黄、橙色、天蓝）
```

### Costume Consistency Checklist

```
✓ 主色调：[具体颜色名称]
✓ 上衣：[类型 + 颜色 + 细节]
✓ 下装：[类型 + 颜色 + 长度]
✓ 配饰1：[类型 + 颜色 + 位置]
✓ 配饰2：[类型 + 颜色 + 特征]
✓ 鞋子：[类型 + 颜色]
✓ 发型：[具体样式]
✓ 特殊标记：[如有，如肩章、徽章等]
```

## Advanced Techniques

### Parallel Action

```
Segment N:
- 0-6秒：驾驶舱，父亲专注驾驶
- 6-12秒：客舱，母亲和女儿互动

（同时发生的两个场景）
```

### Flashback/Flash-forward

```
Current: 女儿在飞机上
Flashback: 女儿小时候第一次见飞机
Return: 女儿现在的反应
```

### Montage

```
快速连续镜头：
- 3秒：女儿尝试1
- 3秒：女儿尝试2
- 3秒：女儿尝试3
- 3秒：最终成功
```

### POV (Point of View)

```
建立：女儿看向窗外
POV：从女儿视角看到的窗外景色
反应：女儿兴奋的表情
```

## Quality Assurance Patterns

### Consistency Check Template

```
角色一致性：
□ 服装颜色相同
□ 配饰位置一致
□ 发型保持不变
□ 年龄特征匹配

场景一致性：
□ 光线方向一致
□ 时间连续合理
□ 空间关系清晰
□ 道具位置合理

动作一致性：
□ 运动方向连贯
□ 速度节奏匹配
□ 情感状态合理
□ 物理规律正确
```

### Prompt Optimization Checklist

```
□ 使用具体颜色名称（深蓝色 vs 蓝色）
□ 包含镜头角度（全景/中景/近景）
□ 描述光线和氛围
□ 使用动作动词
□ 指定情感关键词
□ 避免模糊描述
□ 保持角色定义一致
□ 时间分段清晰
```
